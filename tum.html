HTML Basic

Para criar um documento HTML, você precisa de um editor de texto simples, como o Bloco de Notas ou Visual Studio Code. Comece abrindo o editor de texto e crie um novo arquivo. Em seguida, digite o código HTML básico:

DICA: Se você está utilizando o VS Code, basta digitar "!" que a base irá aparecer.

                <!DOCTYPE html>
                <html lang="pt-BR">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Meu Primeiro Documento HTML</title>
                </head>
                <body>
                    <h1>Olá, mundo!</h1>
                    <p>Este é o meu primeiro documento HTML.</p>
                </body>
                </html>

<!DOCTYPE html>: Esta tag define o tipo de documento como HTML5. É uma instrução para o navegador sobre a versão do HTML que está sendo usada no documento.

<body>:Esta tag engloba todo o conteúdo visível de uma página HTML. Tudo o que é visível na página é colocado dentro dessa tag.

Cabeçalho HTML

Os títulos HTML são definidos com as tags <h1> até <h6>. A tag <h1> define o título mais importante, enquanto <h6> define o título menos importante.

Veja:

            <h1>Cabeçalho 1</h1>
            <h2>Cabeçalho 2</h2>
            <h3>Cabeçalho 3</h3>
            <h4>Cabeçalho 3</h4>

Imagens HTML

As imagens HTML são definidas com a <img> tag. O arquivo de origem (src), o texto alternativo (alt), width e height são fornecidos como atributos.

<img src="html.png" alt="basichtml" width="104" height="142">

HTML Elements

Elementos HTML são blocos de construção de uma página da web, representados por tags como <p> para parágrafos ou <h1> para títulos. Cada elemento tem uma função específica e pode conter texto, outros elementos ou atributos. Eles definem a estrutura e o conteúdo de uma página web.

Veja alguns exemplos:

        <h1>Meu primeiro título</h1>
        <p>Meu primeiro parágrafo</p>

Observe que a tag SEMPRE é finalizada ao final.

Nota: Alguns elementos HTML não possuem conteúdo (como o elemento <br>). Esses elementos são chamados de elementos vazios. Elementos vazios não possuem tag final!

Elementos Alinhados

Em HTML, os elementos podem ser aninhados, o que significa que um elemento pode conter outros elementos dentro dele. Todos os documentos HTML consistem em elementos HTML aninhados, formando uma estrutura hierárquica que define o conteúdo e a organização da página da web. Observe:

            <!DOCTYPE html>
            <html lang="pt-BR">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Meu Primeiro Documento HTML</title>
            </head>
            <body>
                <h1>Título</h1>
                <p>Este é o meu primeiro parágrafo.</p>
            </body>
            </html>

Utilizamos a junção de várias tags.

HTML Attributes

  • Cada elemento HTML pode ter atributos.
  • Os atributos oferecem informações extras sobre os elementos.
  • Eles são sempre indicados na tag de abertura do elemento.
  • Geralmente, os atributos são definidos em pares nome/valor, como: nome="valor".

    • Atributo "href"

    A <a> tag define um hiperlink. O href atributo especifica o URL da página para a qual o link leva:

    <a>href='https://www.google.com.br/>Acesse ao Google.'</a>

    Acesse ao Google.

    Atributo "src"

    A <img> tag é usada para incorporar uma imagem em uma página HTML. O src atributo especifica o caminho para a imagem a ser exibida:

    <img src="progamação.jpg" alt="basichtml" width="104" height="142">

    Existem duas formas de especificar o URL no atributo src:

    1. URL absoluto: Este tipo de URL aponta para uma imagem externa hospedada em outro site. Por exemplo: `src="https://www.w3schools.com/images/img_girl.jpg"`.

    2. URL relativo: Este tipo de URL aponta para uma imagem hospedada no mesmo site. Aqui, o URL não inclui o nome de domínio. Se o URL começar sem uma barra, ele será relativo à página atual. Por exemplo: `src="img_girl.jpg"`. Se o URL começar com uma barra, ele será relativo ao domínio. Por exemplo: `src="/images/img_girl.jpg"`.

    Dica: Em muitos casos, é preferível usar URLs relativos. Eles não quebrarão se você mudar de domínio.

    Atributos de largura e altura

    A <img> tag também deve conter os atributos widthe height, que especificam a largura e a altura da imagem (em pixels):

    <img src="estudo.jpg" alt="basichtml" width="450" height="250">

    ATENÇÃO: O termo "alt" é alternativo, ou seja, não é obrigatório. Ele indica a descrição da imagem, caso ocorra algum impasse e não seja reproduzida.

    Atributo "style"

    O style atributo é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

    <p>style="color:#e0720a;">Esse parágrafo é laranja.</p>

    Esse parágrafo é laranja.

    Com esse atributo, "style", você pode alterar o tamanho, alinhas a posição e até mesmo a fonte. Veja:

    <p>style="color: #e0720a; font-size: 20px; text-align: center;"> Esse parágrafo é laranja e formatado.</p>

    Esse parágrafo é laranja e formatado.

    Detalhe: Você pode utilizar o "style", da mesma maneira, para formatar cabeçalhos, como h1.

    Atributo "lang"

    Você deve sempre incluir o <lang> atributo dentro da <html> tag, para declarar o idioma da página web. O objetivo é ajudar os mecanismos de pesquisa e navegadores. O exemplo a seguir especifica o inglês como idioma:

                        <!DOCTYPE html>
                    <html lang="en">
                    <body>
                        ...
                    </body>
                    </html>

    Atributo "title"

    O <title> atributo define algumas informações extras sobre um elemento. O valor do atributo title será exibido como uma dica de ferramenta quando você passar o mouse sobre o elemento:

    <p>title="PARÁGRAFO">Pare o cursor sobre o parágrafo e veja o título.</p>

    Pare o cursor sobre o parágrafo e veja o título.

    HTML Formatting

    Os elementos de formatação foram projetados para exibir tipos especiais de texto:

    • <b> - Texto em negrito

    • <strong> - Texto importante

    • <i> - Texto em itálico

    • <em> - Texto enfatizado

    • <mark> - Texto marcado

    • <small> - Texto menor

    • <del> - Texto excluído

    • <ins> - Texto inserido

    • <sub> - Texto subscrito

    • <sup> - Texto sobrescrito

    Exemplo:

            <b>Texto em negrito.</b>
        <strong>Texto importante.</strong>
        <i>Texto em itálico.</i>
        <em>Texto enfatizado.</em>
        <mark>Texto marcado.</mark>
        <small>Texto menor.</small>
        <del>Texto excluído.</del>
        <ins>Texto inserido.</ins>
        <sub>Texto subscrito.</sub>
        <sup>Texto sobrescrito.</sup>

    Agora, veja o resultado de todos esses elementos reunidos:

    Texto negrito.

    Texto importante.

    Texto itálico.

    Texto enfatizado.

    Texto marcado.

    Texto menor.

    Texto excluído.

    Texto inserido.

    Texto subscrito.

    Texto sobrescrito.

    HTML Quotations

    Cotação HTML e elementos de citação

    Neste capítulo, examinaremos os elementos <blockquote>, <q>, <abbr>, <address>, <cite> e <bdo> HTML.

    <blockquote>: define uma seção citada de outra fonte.

    Acompanhe o código:

    <blockquote cite="http://www.worldwildlife.org/who/index.html">
        Por 60 anos, o WWF trabalha para ajudar as pessoas e a natureza a prosperarem. Como a principal organização de conservação do mundo, o WWF atua em quase 100 países. 
        Em todos os níveis, colaboramos com pessoas ao redor do mundo para desenvolver e oferecer soluções inovadoras que protegem comunidades, vida selvagem e os lugares onde vivem.
    </blockquote>

    Você pode estilizar no CSS da seguinte forma:

    blockquote {
        background-color: #f9f9f9;
        border-left: 5px solid #ccc;
        margin: 0;
        padding: 10px 20px;
        font-style: italic;
    }

    Veja como ficou:

    Aqui está uma citação do site da WWF:

    Por 60 anos, o WWF trabalha para ajudar as pessoas e a natureza a prosperarem. Como a principal organização de conservação do mundo, o WWF atua em quase 100 países. Em todos os níveis, colaboramos com pessoas ao redor do mundo para desenvolver e oferecer soluções inovadoras que protegem comunidades, vida selvagem e os lugares onde vivem.

    <q>: define uma citação curta.

    <abbr>: define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

    <address>: define as informações de contato do autor/proprietário de um documento ou artigo.

    <cite>: define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

    <bdo>: BDO significa Substituição bidirecional.
