HTML Basic

Para criar um documento HTML, você precisa de um editor de texto simples, como o Bloco de Notas ou Visual Studio Code. Comece abrindo o editor de texto e crie um novo arquivo. Em seguida, digite o código HTML básico:

DICA: Se você está utilizando o VS Code, basta digitar "!" que a base irá aparecer.

                <!DOCTYPE html>
                <html lang="pt-BR">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Meu Primeiro Documento HTML</title>
                </head>
                <body>
                    <h1>Olá, mundo!</h1>
                    <p>Este é o meu primeiro documento HTML.</p>
                </body>
                </html>

<!DOCTYPE html>: Esta tag define o tipo de documento como HTML5. É uma instrução para o navegador sobre a versão do HTML que está sendo usada no documento.

<body>:Esta tag engloba todo o conteúdo visível de uma página HTML. Tudo o que é visível na página é colocado dentro dessa tag.

Cabeçalho HTML

Os títulos HTML são definidos com as tags <h1> até <h6>. A tag <h1> define o título mais importante, enquanto <h6> define o título menos importante.

Veja:

            <h1>Cabeçalho 1</h1>
            <h2>Cabeçalho 2</h2>
            <h3>Cabeçalho 3</h3>
            <h4>Cabeçalho 3</h4>

Imagens HTML

As imagens HTML são definidas com a <img> tag. O arquivo de origem (src), o texto alternativo (alt), width e height são fornecidos como atributos.

<img src="html.png" alt="basichtml" width="104" height="142">
