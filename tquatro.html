HTML Form Elements

Este capítulo descreve todos os diferentes elementos do formulário HTML. Veja a seguir o uso de cada um deles:

Elemento <input>: ser exibido de diversas maneiras, dependendo do type atributo.

Exemplo:

        <label for="fname">First name:</label>
        <input type="text" id="fname" name="fname">

Resultado:




Elemento <label>: O elemento <label> em HTML é usado para associar um rótulo a um elemento de formulário, como input, textarea, select, etc. Ele fornece uma descrição para o elemento de formulário, melhorando a acessibilidade e a usabilidade do formulário.

Elemento <select>:define uma lista suspensa:

        <label for="cars">Escolha um carro:</label>
        <select id="cars" name="cars">
              <option value="volvo">Volvo</option>
              <option value="saab">Saab</option>
              <option value="fiat">Fiat</option>
              <option value="audi">Audi</option>
        </select>

O <option> define uma opção que pode ser selecionada.

Elemento <textarea>: define um campo de entrada multilinha (uma área de texto):

<textarea name="message" rows="2" cols="30">
            O gato está brincando no jardin.

HTML Canvas

O elemento HTML <canvas> é usado para desenhar gráficos em uma página da web.

Veja como desenhar um círculo:

            <canvas id="myCanvas" width="200" height="100" style="border:1px solid #d3d3d3;"></canvas>
            
            <script>
            var c = document.getElementById("myCanvas");
            var ctx = c.getContext("2d");
            ctx.beginPath();
            ctx.arc(95,50,40,0,2*Math.PI);
            ctx.stroke();
            </script>

HTML SVG

O que é SVG?

  • SVG significa gráficos vetoriais escaláveis
  • SVG é usado para definir gráficos baseados em vetores para a Web
  • SVG define gráficos em formato XML
  • Cada elemento e atributo em arquivos SVG pode ser animado
  • SVG é uma recomendação do W3C
  • SVG integra-se com outros padrões, como CSS, DOM, XSL e JavaScript

Estrela SVG

Veja abaixo como fazer uma Estrela SVG:

        <svg width="300" height="200">
            <polygon points="100,10 40,198 190,78 10,78 160,198"
            style="fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;" />
        </svg>
